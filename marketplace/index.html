Below is an overview of the setup requirements:

High-End Encoder server:

HP DL360G6 / DL380G6 / DL360G7/ DL 380 G7 dual Quad Xeon processor or higher

Mid-End Encoder servers:

HP DL360 G5 / G6 /G7 or DL380 G5 / G6 /G7 Single Quad Xeon processor or higher

Dell Power Edge R300 Single Quad Xeon processor

Low-End Encoder Tower PC:

Dell OptiPlex 980 Desktop

Network Requirements:

  • Broadband service requires 150% of the required broadcasting bit rate. For example 1Mb channel requires 1.5 Mb upload.
  • A public IP address accessible from any Internet connection
  • If there is an active FW than the system requires a static NAT
  • Allow TCP traffic to port 25793/80
  • Static mapped UDP ports 22556 & 22567 to the Broadcaster on the firewall public IP Address (port forwarding inbound/outbound).
  • Assigned IP address per machine (not DHCP)
  • Allow Remote access to the machines (LogmeIn / RDP/ VNC or simular)
